# TDD (Test-Driven Development) Phases for PublicViewKudos Feature

This folder contains the three main phases of the TDD cycle for the PublicViewKudos feature:

- **red/**: Test files (Vitest/Jest and Spring Boot) representing the "red" phase (tests before implementation).
- **green/**: Implementation files and passing tests representing the "green" phase (feature code and passing tests).
- **refactor/**: Supporting configuration and refactor changes.
